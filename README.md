# Cybersecurity Chatbot

This is a simple C# console application that acts as a cybersecurity awareness chatbot.

## Features
- Greets the user with audio
- Asks for user name
- Provides cybersecurity tips and interaction
- Uses SoundPlayer for audio playback

## Technologies Used
- C#
- .NET 8.0
- System.Media

## How to Run
1. Open project in Visual Studio
2. Build solution
3. Run the program
